# Userbot Modules

A collection of custom modules for Hikka, Heroku, and GeekTG userbots.

## Overview

This repository contains self-written modules designed for use with
Hikka, Heroku, and GeekTG-based userbots.

Modules are provided as-is. Some are stable, some are experimental.
If something breaks — congratulations, you’re a tester now.

## Notes

- Modules may depend on specific userbot features.
- Backward compatibility is not guaranteed.
- Expect breaking changes.

## Contributing

Pull requests are welcome if you know what you’re doing.
Garbage PRs will be ignored or closed without explanation.
